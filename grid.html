Grid Layout

O CSS Grid Layout Module oferece um sistema de layout baseado em grade, com linhas e colunas, tornando mais fácil projetar páginas da web sem ter que usar flutuadores e posicionamento.

Esse grid layout contem seis colunas e três linhas, se você usar o inspecionar do seu navegador você consiguirar enxergar os grids clicando no elemento HTML que possue a class 'grid-container', e atráves dela poderá alterar a proporção de cada item.
