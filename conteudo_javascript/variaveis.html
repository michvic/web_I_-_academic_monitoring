Variaveis

Assim como em outras liguagens de programação com o javascript é possível trabalhar com variaveis para armazenar valores em memória e também condicionais para controle de fluxo.

Abra o web console e coloque os comandos abaixo. No exemplo temos um exemplo simples de como declarar uma variavel usando var, let ou const.

      
        var exemplo_var = 'Hello'
        let exemplo_let = ' World'
        const exemplo_const = ' Student'

É importante lembrar que existem diferenças de contexto entre estas 3 formas de declarar uma variavel. Sendo var uma forma de escopo mais geral podendo ela ser lida em diferentes contextos inclusive em escopo fechado, let tem o escopo mais fechado limitando-se a este, e const também de escopo geral porém seu valor não pode ser alterado depois da sua declaração.
