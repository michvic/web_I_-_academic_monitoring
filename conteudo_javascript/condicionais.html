Condicionais

Para controlar o fluxo dentro de um programa de computador são necessárias estruturas para prover isso em uma linguagem de programação.

No javascript a estrutura basica é apresentada abaixo. O if é a condição 'se' algo for verdadeira alguma coisa acontece.

      
        if (/* condicao */) {
          /* código a ser executado caso a condicao seja verdadeira */
        }

A condição else é o 'caso contrario', qual a estrutura base é apresentada abaixo.

      
        if (/* condicao */) {
          /* código a ser executado caso a condicao seja verdadeira */
        } else {
          /* código a ser executado caso a condicao seja falsa */
        }

A condição é switch é uma forma de ter uma condição testada diversas vezes.

      
        var tinta = 'azul';
        switch (tinta) {
          case 'azul':
            console.log('Irei pintar o carro de azul');
            break;
          case 'amarela':
            console.log('Vou pintar a casa de amarelo');
            break;
          case 'verde':
            console.log('Vou pintar o chão da garagem de verde');
            break;
          default:
            console.log('Não vou pintar nada');
            break;
        }
