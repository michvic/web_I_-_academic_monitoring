Arrays e Loops

Quando necessitamos de armazenar mais de um valor em uma estrutura são necessárias estruturas que possibilitem o armazenamento destes valores. Neste tópico discutiremos estas estruturas.

Um exemplo de declaração de um array básico é apresentada abaixo.

      
        let valores = [10,20,30];

Abaixo temos uma demonstração de como acessar valores dentro de um array pela sua posição.

      
        valores[1]
        /* este retornará a o valor da segunda posição, ou seja, o valor 20 */

for

Agora veremos um exemplo básico de loops que são estruturas de repetição do javascript. Aproveitando nosso exemplo de array vamos iterar sobre o array declarado anteriormente com o nosso primeiro exemplo.

      
        for(let i = 0; i <= valores.length; i++){
          console.log(i, valores[i])
        }

for in

Também temos a forma de acesso for in quando queremos interar sobre os elementos de um array.

      
        for(i in valores){
          console.log(i)
        }

for each

A estrutura for each é geralmente utilizada quando é necessário iterar sobre as propriedades de um objeto.

      
        var valores = [10,20,30];
        valores.forEach(function(i){
          console.log(i);
        });

while

Funciona basicamente igual ao for, e é possível sempre trocar o for() por um while(). Escolhemos um ou outro pela clareza do que estamos fazendo. Geralmente preferimos utilizar o loop for() para interar com contadores e loops while() até que alguma condição mude (de true para false, por exemplo).

      
        var x = true;
        while(x) {
          console.log('Terminando condição verdadeira');
          x = false;
        }

do while

Segue o mesmo princípio do while, mas o corpo é sempre executado pelo menos uma vez, independente da condição, pois primeiro ele faz do e depois testa a condição.

      
        do {
          console.log('Finalizando do while');
        } while(false);

Apesar da condição já começar como falsa, veremos a string "Finalizando do while" uma vez no console do browser.
