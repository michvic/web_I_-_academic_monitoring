Fundamentos do Javascript

O Javascript é uma linguagem de programação client-side utilizada para controlar HTML e CSS para manipular comportamentos de pagina, ou seja, quando vemos algum tipo de animação ou comportamento especial em um site muito provavelmente existe um código javascript por trás deste comportamento.

Vamos a um primeiro exemplo de código javascript. Primeiro abra o web console do navegador qual você está acessando essa pagina. No Firefox o atalho é ctrl + shift + K e no Chrome ctrl + shift + J. Em seguida digite o código abaixo:

      
        console.log('Hello World')
