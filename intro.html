HTML - Hyper Text Markup Language

  • Tag: É o marcador usado para delimitar o início e, onde for requerido, o fim de um elemento HTML. A Tag é identificada por uma palavra ou letra entre os símbolos “< “ e “>”.
  • Atributos: definem propriedades específicas (formatação, identificação ou ação) para o elemento html.
  • Elemento html: É uma estrutura semântica, composta de: tag de abertura, conteúdo e tag de fechamento.
